
# Proyecto AquaLimpia

## Objetivo

En este trabajo se buscó analizar el funcionamiento de las plantas de tratamiento de AquaLimpia para revisar problemas relacionados con la DBO y la eficiencia del tratamiento. Tambien la idea fue usar datos reales para apoyar mejor la toma de decisiones.

---

## Proceso

Primero se cargó el archivo Excel en Python usando Pandas. Despues se revisaron los datos y se eliminaron algunos registros vacios para evitar errores en el analisis.

Luego se calculó la eficiencia de tratamiento comparando la DBO de entrada y salida de cada planta. Tambien se agruparon los resultados para sacar promedios y poder comparar las plantas entre si.

Despues se hizo un grafico simple con Matplotlib para visualizar mejor los resultados y entender mas facil las diferencias entre plantas.

Finalmente se organizó el trabajo en carpetas separadas para mantener mas ordenados los datos, scripts y reportes del proyecto.

---

## Resultados

Con el analisis se pudieron observar diferencias de eficiencia entre las plantas de tratamiento. Algunas presentaron mejores resultados y otras mostraron valores mas bajos.

El grafico ayudó a entender los resultados de forma mas rapida y visualizar mejor el comportamiento general de las plantas.
